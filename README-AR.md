# موقع MasterCleanNord

موقع ثابت ومتجاوب باللغة الألمانية، مصمم ليناسب شعار MCN MasterCleanNord.

## الملفات
- `index.html` الصفحة الرئيسية
- `styles.css` التصميم والاستجابة للجوال
- `script.js` القائمة، الحركات ونموذج التواصل
- `assets/` الشعار والأيقونة
- `impressum.html` صفحة بيانات الشركة (تحتاج استكمال البيانات القانونية)
- `datenschutz.html` قالب سياسة الخصوصية (يحتاج مراجعة قانونية وتعديل حسب الاستضافة)

## التشغيل
افتح `index.html` مباشرة، أو ارفع المجلد كاملًا إلى الاستضافة.

## قبل النشر
1. أكمل بيانات المدير والسجل التجاري والرقم الضريبي في `impressum.html`.
2. عدّل سياسة الخصوصية حسب شركة الاستضافة وأي أدوات تحليل أو خرائط ستضيفها.
3. النموذج الحالي يفتح برنامج البريد ويرسل إلى `info@mastercleannord.de`. لربطه بقاعدة بيانات أو PHP يلزم إضافة Backend.
4. راجع أرقام الهاتف والعنوان والبريد قبل النشر النهائي.
